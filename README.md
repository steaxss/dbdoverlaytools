## üìñ Quick Start Guide

### First Time Setup
1. Create a premium account at [dbdoverlaytools.com](https://dbdoverlaytools.com)
2. Download and install the application
3. Log in with your credentials
4. Select your desired overlay type
5. Click the switch icon to show the overlay
6. Position and customize as needed

### Adding to OBS/Streamlabs
1. Add a new **Window Capture** source
2. Select the overlay window (e.g., "DBD OverlayTools - Timer Overlay")
3. Position in your scene
4. Lock the overlay in the app settings and disable/enable always on top depending on your needs

## üîß Troubleshooting

### Can't log in
- Verify you have a premium account
- Check your internet connection

### Overlay not showing in OBS
- Make sure the overlay window is open (switch icon enabled)
- Use **Window Capture**
- Try restarting OBS
